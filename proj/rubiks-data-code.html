using the gan 6x6 rubik's cube mosaic set as a scannable code to my homepage

a couple days ago, i felt the sudden urge to clean up my desk. in the midst of all the egregious clutter, my eyes were drawn to...

the GAN Mosaic Cube 6x6

i'd completely forgot about it and hadn't touched for so long that it was collecting dust on the top. wanting to give it new purpose, i decided to make it a qr code that links to my homepage. the mosaic set that i have is 6 cubes by 6 cubes which is 18x18 pixels, or definitely not enough for a qr code. so i did some research on two-dimensional barcodes, and decided to use a data matrix code bc it was the only widespread code that fit.
