[project id #001] using the gan 6x6 rubik's cube mosaic set as a scannable code for my homepage

a couple days ago, i felt the sudden urge to clean up my desk. in the midst of all the clutter, i rediscovered...

my GAN rubiks cube mosaic set!!!

i'd completely forgot about it and hadn't touched for so long that it was collecting dust on the top. wanting to give it new purpose, i had the brilliant idea to make it a scannable code that links to this website. the set that i have is 6 cubes by 6 cubes or 18x18 "pixels", which wasn't enough space for a qr code. after a quick google search, i chose to make a data matrix code because it was the only widespread code that fit. due to each "pixel" having a black border, the blue and yellow color scheme i used was the only color combination that my phone could recognize. even so, it sadly does not scan easily all the time, but it looks very cool. here's a picture of the final product on my desk (click on the image to view at full size):

note that the default camera app doesn't scan data matrix codes (at least on android) so you need to use a third party app. i used google lens.
